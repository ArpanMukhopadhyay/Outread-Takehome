# Outread-Takehome

This script parses a number of academic journals in pdf form and clusters them based on abstract similarity using a standard K-means algorithm.

Dependencies:
`
    - numpy=1.26.2
    - pandas=2.1.4
    - pypdf2=2.11.1
    - scikit-learn=1.3.2
    - scipy=1.12.0
    - matplotlib-base=3.8.2
    - nltk=3.8.1
    - pymupdf==1.24.7
`

To run the script
`python cluster.py`

Inspiration of approach drawn from:
